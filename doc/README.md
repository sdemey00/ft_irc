## IRSSI client usage
```
/set nick <new nickname>
/set dcc_autoget
/set dcc_download_path <directory>
/connect <ip address> <port number>(6667) [password]
/window list
/window <number>
/msg <user> <message>
/dcc {send, get} <user> <filename>
/dcc close send <user>
/join <channel name>
/part
/disconnect
```
